Esta API foi desenvolvida com o intuito de criar meu primeiro contato com o desenvolvimento de sistemas, nela, utilizei java como linguagem de desenvolvimento, bem como springboot para o framework e Postgres para a geração e manipulação do banco de dados.
Elencado ao uso dos softwares, pude praticar os conceitos e métodos que compõem um CRUD, usei entidades de Mapeamento e criei logicas que permitissem prever exceções como a leitura de um id que não foi cadastrado e etc.
Apesar do sistema ja apresentar certa complexidade, grande parte dos conceitos que utilizei para a construção do sistema ainda são muito novos e desconhecidos para mim. Como proximo passo para o meu desenvolvimento, viso a concepção de um sistema sem o auxilio externo de guias e moldado em uma ideia original minha.
